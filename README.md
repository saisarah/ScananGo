# Scanan-Go

## Description
Scanan-Go is a Contact Tracing System that records individuals visiting certain establishments.

### Technologies Used
- C#.Net (Winforms)
- SQL Database
- GUNA2UI
- SMTP Server

### Features
 - Admin Login
 - Admin Panel
 - Add Admin
 - Delete Admin
 - Generates QR Code
 - Scanning QR Code
 - Sending of QR Code to Email
